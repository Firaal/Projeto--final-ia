Classificação de Consumidor de bebidas alcoólicas

Bem-vindo ao BebaAI

Este projeto utiliza a Inteligência Artificial para responder a uma pergunta importante: Como classificar indivíduos como bebedores ou não bebedores com base em dados? Combinando ciência de dados e tecnologia avançada, criamos um modelo de IA capaz de realizar essa classificação de forma precisa e eficiente.


Explore análises detalhadas, gráficos informativos e insights valiosos que mostram como os dados podem revelar padrões interessantes e ajudar na tomada de decisões. Aqui, você encontrará uma abordagem prática e explicativa sobre todo o processo.


Descubra como a IA está transformando informações em soluções inteligentes!

Contexto do Estudo

Este estudo busca explorar dados relacionados aos hábitos de consumo de álcool e cigarro, com o objetivo principal de classificar indivíduos como bebedores ou não bebedores. Embora a base de dados inclua informações sobre o status de fumante, essas serão utilizadas como um atributo auxiliar para ajudar na precisão da classificação.


A análise foi realizada utilizando técnicas de Análise Exploratória de Dados (EDA), permitindo identificar padrões de comportamento e relações importantes entre as variáveis. Esses insights foram essenciais para desenvolver um modelo de Inteligência Artificial que realiza a classificação de forma eficiente e precisa.
